'Разделяй и властвуй' - это не определённый алгоритм решения, а метод решения задач с помощью рекурсии.
Суть данного метода состоит в том, чтобы разделить задачу на меньшие подзадачи. После этого подзадачи рекурсивно вычисляются.
Результаты вычисления каждой подзадачи при использовании данного метода не сохраняются. А в динамическом программировании наоборот.
Пример:
Задание: Найти числа Фибоначчи.
Решение с помощью алгоритма 'Разделяй и властвуй':
def fib(n):
    if n < 2:
        return 1
    else:
        return fib(n - 1) + f(n - 2)
Динамическое программирование:
mem = []
fib (n)
    if n in mem:
        return mem[n]
    else:
        if n < 2:
            f = 1
        else:
            f = fib(n - 1) + f(n - 2)
        mem[n] = f
        return f
Преимущества алгоритма:
- Идеально подходт для многоядерных систем.
- Эффективно расходует кэш-память.
- Сложность данного алгоритма меньше, чем у примитивного метода решений.
Сложность умножения двух матриц с помощью примитивного метод - O(n^3). А сложность метода 'разделяй и властвуй' - O(n^2.8)
Где применяется:
- Бинарный поиск
- Сортировка слиянием
- Быстрая сортировка
- Алгоритм Карацубы
- Умножение матриц методом Штрассена
