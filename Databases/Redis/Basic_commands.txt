Основные команды:
redis-cli - запуск клиента redis

SET - сохранить значение в ключе.
SET name "Peter"
OK

GET - извлечь значение:
GET name
"Peter"

EXISTS - проверить существование ключа.
EXISTS name
(integer) 1
Возвращает 1, если ключ существует, и 0 в противном случае.

EXPIRE - можно установить срок истечения ключа, она позволяет устанавливать время жизни ключа в секундах.
Ещё одним вариантом является команда EXPIREAT, которая на входе ожидает отметку времени в формате Unix.
!!!Срок истечения ключа общепринято использовать, когда Redis применяется в качестве кеша или для хранения волатильных данных.
EXPIRE name 2
(integer) 1
Спустя 2 секунды:
GET name
(nil) - нулевой ответ, ключ не найден.

DEL - удаляет ключ.
