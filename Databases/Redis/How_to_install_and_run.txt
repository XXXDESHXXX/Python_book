1. Получить образ Redis платформы Docker:
docker pull redis

2. Исполнить следующую команду в оболочке, чтобы запустить контейнер Docker для Redis.
docker run -it --rm --name redis -p 6379:6379 redis
Опции этой команды:
-it сообщает, что можно переходить прямо внутрь контейнера для ввода.
--rm сообщает Docker, что при выходе из контейнера нужно очищать контейнер и удалять файловую систему автоматически.
--name используется для назначения контейнеру имени.
-p применяется для публикации порта 6379, на котором работает Redis, на тот же порт интерфейса хоста.

3. Запустившийся сервер Redis нужно оставить открытым. И выполнить следующую команду:
docker exec -it redis sh
Вы запустили клиент Redis!
4. Запустить клиент Redis
redis-cli
Готово!

5. Для запуска Redis c питоном необходимо установить redis-py следующей командой:
pip install redis
