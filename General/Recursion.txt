Рекурсия - вызов функции самой себя
Пример рекурсии - вычисление факториала
def factorial_recursive(n):
    if n == 1:
        return n
    else:
        return n*factorial_recursive(n-1)
Рекурсия не является лучшим методом для решения задач. Циклы могут ускорить работу программы, рекурсия может ускорить работу программиста
Для вызова различных функций исплользуется стек вызовов.
В стеке вызовов помещается вызов каждой функции в программе.
Из стека можно либо удалять элементы, либо добавлять новые.
Для окончания рекурсии используется базовый случай рекурсии:
    if n == 1:
        return n
Сама же рекурсия называется рекурсивным случаем:
    else:
        return n*factorial_recursive(n-1)
