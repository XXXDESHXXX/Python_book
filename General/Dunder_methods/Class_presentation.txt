Представление своих классов
Str
Определяет поведение функции str(), вызванной для экземпляра класса.

Repr
Определяет поведение функции repr(), вызванной для экземляра вашего класса. Главное отличие от str() в целевой аудитории.
repr() больше предназначен для машинно-ориентированного вывода, а str() предназначен для чтения людьми.

Format
Определяет поведение, когда экземпляр вашего класса используется в форматировании строк нового стиля.

Hash
Определяет поведение функции hash(), вызванной для экземпляра вашего класса. Возвращает целочисленное значение, которое будет
использоваться для быстрого сравнения ключей в словарях.

Bool
Определяет поведение функции bool(), вызванной для экземпляра класса. Возвращает True или False, в зависимости от того, когда
вы считаете экземпляр соответствующим True или False.

Dir
Определяет поведение функции dir(), вызванной на экземпляре вашего класса. Этот метод будет возвращать пользователю список атрибутов.

Sizeof
Определяет поведение функции sys.getsizeof(), вызванной на экземпляре вашего класса. Метод должен вернуть размер вашего объекта в байтах.
