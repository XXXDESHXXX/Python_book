Декоратор - это паттерн программирования, который позволяет менять поведение функции/кода не меняя саму реализацию этой функции
Следует понимать, что:
1) Функция - полноправный объект, который можно передавать, возвращать, хранить в коллекции и делать всё то, что с другими
объектами.
2) Внутренняя функция может захватывать значения переменных из внешней
def summ(a, b):
    return a + b

example = summ # в данном примере создаётся объект в памяти(НЕ РЕЗУЛЬТАТ ФУНКЦИИ)
def summ(a, b):
    return a + b

example = summ(1, 3) # мы получим уже не ссылку на объект в памяти, а результат сложения чисел!
