Список - упорядоченная, изменяемая коллекция объектов произвольных типов.
Создать список можно встроенной функцией list().
Пример создания списка:
list (' список') [' с', 'п', 'и', 'с', 'о', 'к'].
Переменная, определяемая как список, содержит ссылку на структуру в памяти, которая в свою очередь хранит ссылки на какие-либо другие
объекты
или структуры.
Типы списков:
- пустой список.
Примеры:
a = []
a = list()
- Список с заранее заданным набором данных.
a = [1, 2, 3]
- Список списков:
people = [
    ['Tom', 29],
    ['Alice', 33],
    ['Bob', 27]
]
Копию списка можно создать следующими способами:
- b = a[:]
- b = list(a)
Примечание!
Если выполнить присвоение списка b списку a, то переменной b будет присвоена ссылка на тот же элемент данных в памяти, на который
ссылается a, а не копия списка. Т.е. если список a будет изменяться, то и b будет изменяться тоже.
Индексация в списке начинается с 0.
Список можно разложить на элементы:
people = ['Tom', 'Bob', 'Sam']
tom, bob, sam = people
print(bob)
out: Bob
Списки можно объединять:
Для объединения списков применяется операция сложения (+):
people1 = ['Tom', 'Bob', 'Alice']
people2 = ['Tom', 'Sam', 'Tim']
people3 = people1 + people2
