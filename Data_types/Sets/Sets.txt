Множество - это неупорядоченная последовательность элементов, каждый из которых представлен ровно один раз.
Множество хранит только hashable значения.
Способы создания множеств:
1. С помощью функции set():
users = set()
print(users)
out: {}
В set() также можно передавать списки или кортежи элементов. Они будут конвертироваться во множество.
2. Явная инициализация:
users = {'Tom', 'Bob', 'Alice', 'Tom'}
print(users)
out: {'Alice', 'Bob', 'Tom'}
В данном случае множество вывело только один элемент Tom, так как множество содержит только уникальные значения.
Перебор элементов множества осуществляется при помощи цикла for.
Существует также тип frozenset, который является видом множества, которое не может быть изменено. Для его создания
используется функция frozenset.
В функцию frozenset передаётся набор элементов - список, кортеж, другое множество.
Во множество, образованное таким образом нельзя добавить новые элементы, как и удалить из него уже имеющиеся.
