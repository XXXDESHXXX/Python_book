Существуют различные методы для работы с кортежами.
Основными среди них являются:
- len()
Возвращает длины кортежа.
- sum()
Возвращает сумму элементов кортежа.
- min()
Возвращает минимальный элемент кортежа.
- max()
Возвращает максимальный элемент кортежа.
- count()
nums = (1, 2, 3, 4, 1, 1,)
print(nums.count(1))
out: 3
Подсчитывает число вхождений какого-либо значения в кортеже.
- sorted()
nums = (4, 1, 2, 3, 5, 0)
print(sorted(nums))
out: [0, 1, 2, 3, 4, 5]
Сортировка элементов кортежа, однако результатом будет список.
Для сохранения результата в виде кортежа, следует использовать функцию tupple().
nums = (4, 1, 2, 3, 5, 0)
print(tuple(sorted(nums)))
out: [0, 1, 2, 3, 4, 5]
Кортежи возможно преобразовать в другие типы данных:
- строка
nums = (1, 2, 3)
print(''.join(nums))
out: 123
- список
nums = (1, 2, 3)
print(list(nums))
out: [1, 2, 3]
- словарь, только если кортеж вложенный и состоит из пар значений
info = (('Fruits', 5), ('Vegetables', 15), ('Candies', 3))
print(dict(info))
- множество
nums = (3, 2, 1, 6)
print(set(nums))
{3, 2, 1, 6}
Кортежи можно сравнивать между собой, с помощью стандартных операторов сравнения.
Если элементы кортежей принадлежат к разным типам данных, поддерживаются только операторы == и !=.
