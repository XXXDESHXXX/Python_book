Кортеж - это неизменяемый, упорядоченный тип данных.
В кортеж нельзя добавить, удалить или изменить его элементы.
Чаще всего кортеж хранит значения разных типов.
Способы создания кортежей:
- с помощью функции tuple():
tom = tupple()
out: ()
- с заранее заданными параметрами:
tom = ('Tom', 23)
out: ('Tom', 23)
- без использования скобок:
tom = 'Tom', 23
out: ('Tom', 23)
Если кортеж состоит из одного элемента, то после единственного элемента необходимо поставить запятую.
tom = ('Tom',)
Обращение к элементам кортежа:
Обращение к элементам кортежа происходит по индексу. Индексация начинается с 0. При указании индекса равному -1 будет
выбран последний элемент кортежа.
При необходимости можно разложить кортеж на отдельные переменные:
name, age, company, position = ('Tom', 37, 'Google', 'software developer')
print(name)
out: Tom
С помощью среза можно получить часть кортежа в виде другого кортежа:
tom = ('Tom', 37, 57, 67)
print(tom[1:3])
out: 37, 57
Примечание!
Кортежи следует применять, когда необходимо вернуть несколько значений из функции. Когда функция возвращает несколько
значений, фактически она возвращает кортеж.
Важные особенности кортежей:
1. Кортежи поддерживают неограниченный уровень вложенности:
numbers = ((4, 5, 8, (5, 1, (5, 3))), (7, 2, (4, 5, (3, 1, 1))))
2. Кортежи работают немного быстрее, чем списки - это связано с особенностями хранения кортежей в памяти.
3. Кортежи занимают меньше места в памяти.
4. Кортежи поддерживают конкатенацию + и повторение * n.
5. Кортежи можно распаковать.
seasons = ('весна', 'лето', 'осень', 'зима')
s1, s2, s3, s4 = seasons
print(s1, s3)
out: весна осень
