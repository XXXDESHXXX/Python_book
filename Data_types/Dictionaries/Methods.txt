Существуют различные методы для работы со словарями:
- get(key, default)
Данный метод возвращает из словаря элемент с ключом key. Если элемента с таким ключом нет, то возвращает значение
по умолчанию default.
- del
del users[key]
Данный метод удаляет элемент по ключу, если такого ключа нет то будет выведена ошибка KeyError.
- pop(key, default)
Метод pop удаляет элемент по ключу и возвращает удалённый элемент. Если элемент отсутствует, то возвращается значение
default.
- clear()
users.clear()
Данный метод удаляет все элементы в словаре.
- copy()
Копирует содержимое словаря, возвращая новый словарь.
- update()
Объединяет два словаря
users = {"+1111111": "Tom", "+3333333": "Bob"}

users2 = {"+2222222": "Sam", "+6666666": "Kate"}
users.update(users2)
Примечание!
При этом словарь users2 остаётся без изменений.
- items()
users = {
    "+11111111": "Tom",
    "+33333333": "Bob",
    "+55555555": "Alice"
}
for key, value in users.items():
    print(f"Phone: {key}  User: {value} ")
Данный метод используется при переборе значений в словаре. Он возвращает набор кортежей. Каждый кортеж содержит ключ и
значение элемента, которые можно тут же получит в переменные key и value.
- keys()
Этот метод используется для перебора ключей
- values()
Этот метод используетя для перебора значений
