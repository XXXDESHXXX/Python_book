Сравнение строк в python имеет свои отличия и особенности.
В python для сравнения строк используют следующие операторы:
- Оператор <, 'меньше'
- Оператор <=, 'меньше или равно'
- Оператор ==, 'равно'
- Оператор !=, 'не равно'
- Оператор >, 'больше'
- Оператор >=, 'больше или равно'
При сравнении строк в python сравнивается их значения в таблице Unicode.
Пример:
pring('apple' > 'Apple')
out: True
Так как латинская 'A' имеет значение 65, а строчная 'a' имеет значение 97.
Для того, чтобы узнать значение какого-нибудь символа, можно использовать функцию ord():
print(ord('A'))
out: 65
При сравнении символов или строк, Python конвертирует символы в их соответствующие порядковые значения, после чего сравнивает слева направо.
Также есть функция chr(), преобразовывающая порядковое значение в символ.
print(chr(1040))
out: A
Рекомендуется сравнивать строки одного регистра! Для этого можно использовать методы lower() и upper().
