Интернационализация - это процесс адаптации ПО под потенциальное использование разных языков и локалей, чтобы избежать его привязки к
определённому языку или локали.
Локализация - это процесс фактического перевода ПО и его адаптации под конкретную локаль.'
Файл сообщения - это обычный текстовый
Для локализации django имеет следующие команды, служащие для управления переводами:
makemessages: пробегает по дереву исходного кода, чтобы отыскать все строковые литералы, помеченные для перевода, и создаёт или обновляет файлы
сообщений .po в каталоге locale. По каждому языку создаётся один файл .po
compilemessages: компилирует существующие файлы сообщений .po в файлы .mo, которые используются для извлечения переводов.
