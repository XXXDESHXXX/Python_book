1.Сначала следует установить Django, для этого следует прописать команду:
pip install Django~=4.1.0
2.Далее следует проверить успешность установки Django:
python -m django --version
Если вы получили результат 4.1.X, то, значит, Django был успешно установлен, иначе вы получите сообщение No module named Django.
3. Для создания проекта нужно выполнить следующую команду в командной оболочке:
django-admin startproject mysite
Проеряем миграции выполняя следующую команду:
python manage.py migrate
Везде должно быть OK.
4. Запускаем сервер разработки, набрав следующую команду:
python manage.py runserver
Теперь пройдите по URL-адресу http://127.0.0.1:8000/ в своем браузере вы увидите страницу, на которой указано, что проект успешно выполняется.
5. Создаём приложение:
python manage.py startapp blog
